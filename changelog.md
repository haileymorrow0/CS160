# Changelog - Checkly Sprint #2 Update

## Hailey Thompson
## March 19, 2026

## [Sprint 2] 

###  Sprint Goal
Finalize the real-time synchronization engine and implement a categorization system to make sure users can easily organize and collaborate on tasks and lists across all devices.

### Increment
For this increment, I am focusing on the organizational backbone of Checkly. While the syncing is happening in the background, the categories and shared lists are what the user interacts with.

### Changelog
- **Category Architecture:** Implemented create, read, update, delete functionality for custom lists. Users can now organize tasks into "Work," "Personal," and "School" categories.
- **Global UI Refresh:** Updated the main dashboard to support a tabbed navigation system, allowing for quick filtering by category.
- **Color-Coding System:** Added visual identifiers for different list types to improve user scannability.

### Work in Progress
- **Cross-Device Sync:** Currently adding connections to ensure tasks updates in real-time across all platforms.
- **Collaborative Shared Lists:** Developing invitation logic and permission-based roles (Viewer vs. Editor) for shared "Team" and "Family" lists.
