# Changelog - Checkly Sprint #5 Update

## Hailey Thompson
## April 10, 2026

## [Sprint 5]

### Sprint Goal
Implement cool and fun "Smart Reminders" notification engine and the "Personal Productivity Dashboard" to provide users with actionable insights and improved notification reliability.

### Increment
For this increment, the focus shifted toward data visualization and the technical "Smart" features of the app. I wanted to move beyond just listing tasks and start showing users their progress over time and visualize it super cool. Adults and kids love seeing progress as they complete tasks.

### Changelog
- **Roles:** Updated the "Collaborator Management" screen to give the "Owner" a unique purple badge and expanded permissions, clearly distinguishing them from "Editors" as requested in Sprint 4 feedback.
- **Smart Reminders:** Successfully implemented the logic for time-sensitive notifications that appear on the mobile lock screen, even when the app is closed.
- **Productivity Dashboard:** Launched a new "Summary" view featuring data visualizations (bar charts and progress rings) that track tasks completed, pending, and overdue.
- **Improved Sync:** Optimized the background synchronization process so shared family lists update in near real-time (under 2 seconds) across multiple devices. (My kids said it was too delayed to skip out on important tasks)

### Feedback Received
- Stakeholders (Husband/Consultant) noted that the distinct "Owner" badge finally clears up the confusion regarding who can delete a shared list versus just editing it.
- My "Junior Stakeholders" (the kids) really liked seeing the progress bars fill up on the dashboard—it makes it feel a bit more like a game.
- Suggested adding a "Dark Mode" in the next sprint to help with nighttime planning.
- Recommendation to add a "Snooze" feature for the Smart Reminders so users aren't overwhelmed by alerts.

### Work in Progress
- **Gamification Layer:** Designing the badge and reward system for consistent task completion.
- **Dark Mode UI:** Creating a high-contrast theme for low-light usage.
- **Voice Integration:** Researching how to add tasks via voice commands for hands-free list management.

### Project Management
- **Tracking Tool:** [Jira](https://byupathway-team-j753k3ol.atlassian.net/jira/software/projects/DEV/boards/1)
- **Current Velocity:** Stable. The WIP limits continue to help me close out features completely before starting new ones.
